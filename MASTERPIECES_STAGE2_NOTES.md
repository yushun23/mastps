# MasterPieces Stage 2 更新说明

本次更新重点是把你在 macOS 调试时遇到的问题固化到代码包里，并推进项目管理逻辑。

## 已修复

1. 删除/备份根目录无效 `Cargo.toml`，避免 Tauri 错读根目录 Rust manifest。
2. 补齐 `src-tauri/tauri.conf.json` 的窗口配置，程序启动后会创建主窗口。
3. 补齐基础图标文件：`32x32.png`、`128x128.png`、`128x128@2x.png`。
4. 精简 Tauri icon 配置，避免继续寻找不存在的 `icns/ico` 文件。

## 新增

1. 项目选择页增加“打开已有项目”。
2. 项目丢失后点击“进入”会提示输入新位置。
3. 右键项目增加“重新定位”。
4. 重新定位时会读取项目内部 `project.json` 的项目代码，项目改名不影响识别。
5. 如果重新定位到另一个项目，会提示是否作为另一个项目加入列表。

## 运行方式

```bash
npm install
npm run tauri:dev
```

如果你正在使用旧目录，可以直接把本包里的文件覆盖到当前 `mastps` 项目目录。
